# Docker.com - Get Started

https://docs.docker.com/get-started/

# Commands

```sh
# Hello world
$ docker run hello-world

# Pull busybox image from docker registry
$ docker pull busybox

# List images
$ docker images

# Run busybox image
$ docker run busybox

# Build docker image
$ docker build -t japchae-api .

#
$ docker run -p 127.0.0.1:3000:3000 getting-started

```
